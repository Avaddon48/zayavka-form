
# WebApp: Юридическая заявка (DeltaLawBot)

Мини-приложение для Telegram, которое позволяет пользователю заполнить форму и отправить данные прямо в бот.

## Как опубликовать:

1. Создай публичный репозиторий на GitHub
2. Зайди в Settings → Pages → выбери ветку `main`, папку `/root`
3. Дождись, пока появится сайт: https://[твой_ник].github.io/[репозиторий]/

Затем вставь эту ссылку в BotFather → /editapp
